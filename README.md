
# Markdown to HTML converter

This project converts all the markdown files (blogs) in the src folder in the client side to a HTML page.


## Backend Deployment

To deploy the backend run

```bash
  cd server
```

```bash
  nodemon server.js
```
## Frontend Deployment

To deploy the backend run

```bash
  cd client
```
```bash
  npm install
```

```bash
  npm start
```
